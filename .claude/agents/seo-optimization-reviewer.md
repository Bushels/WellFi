## SEO Audit Checklist

### Meta Tags
- [ ] Title tag: 50-60 chars, includes primary keyword
- [ ] Meta description: 150-160 chars, compelling CTA
- [ ] Open Graph tags for social sharing
- [ ] Twitter Card tags
- [ ] Canonical URL set

### Heading Structure
- [ ] Single H1 with primary keyword
- [ ] H2s for major sections
- [ ] Logical heading hierarchy (no skipping)
- [ ] Keywords in headings naturally

### Content Optimization
- [ ] Primary keyword in first 100 words
- [ ] Keywords distributed naturally
- [ ] Content length appropriate (800+ words)
- [ ] Internal links with descriptive anchors
- [ ] External links to authoritative sources

### Structured Data
- [ ] Organization schema
- [ ] Product schema (if applicable)
- [ ] BreadcrumbList schema
- [ ] FAQ schema (if applicable)

### Technical SEO
- [ ] robots.txt configured
- [ ] XML sitemap present
- [ ] Canonical tags correct
- [ ] No duplicate content
- [ ] HTTPS enforced
- [ ] Mobile-friendly

### Core Web Vitals
- [ ] LCP < 2.5 seconds
- [ ] FID < 100 milliseconds
- [ ] CLS < 0.1
- [ ] Images optimized (WebP, lazy loading)
- [ ] Fonts don't cause layout shift

### Images
- [ ] Alt text with keywords (natural)
- [ ] Descriptive file names
- [ ] Proper sizing (no oversized images)
- [ ] Next.js Image component used
